Installation ~ Part I

Backup all files before preceding

While it is recommended to test integration of this mod locally using wampServer, xampp or similar, instructions will proceed as though the mod is being applied to a live site; as that's likely what many users will attempt... Part one of installation should go fairly smoothly. Follow each step in the instructions.

 

Upgrading from a Previous Version

If this is a first-time install, skip this section

For those users upgrading from a previous version of Advanced BBCode Box, it is recommended that you delete the previous directories which contained the mod files; due to the large amount of changes and deprecated files. Depending on which version you are using, they could be located at:

mods/bbcode_box
modules/Forums/bbcode_box

If performing this installation on a live site, note that your current version of Advanced BBCode Box will no longer function after deleting these directories, and that a new copy of includes/bbcode_box will be uploaded in the next step. If this is an issue you can wait to upload any files until the mod is completed, but it's up to you to keep track of things.. :)

/**
 * This is used to strip bbcode from with [pre=lang][/pre] tags
 * as they are not run through the same process as [code][/code]
 * tags.  Also needed to keep emoticons from appearing in code 
 * samples. Used within function bbencode_first_pass_pda().
 */
function replace_nestedbbcode($text, $uid)
{
	$stripuid = array(":1:$uid", ":u:$uid", ":o:$uid", ":$uid");
	$text = str_replace($stripuid, '', $text);
	$code_entities_match = array('#<#', '#>#', '#"#', '#:#', '#\[#', '#\]#', '#\(#', '#\)#', '#\{#', '#\}#');
	$code_entities_replace = array('&lt;', '&gt;', '&quot;', '&#58;', '&#91;', '&#93;', '&#40;', '&#41;', '&#123;', '&#125;');
	$text = preg_replace($code_entities_match, $code_entities_replace, $text);
	return $text;
}

Posting Preview Edits

This edit is only required in some themes, making sure that {MESSAGE} is NOT wrapped in an inline element such as <span></span>. This issue is present in the default RN subSilver template, but in none of the other core RN themes.


  • yoursite.com
  • modules
  • Forums
  • templates
  • subSilver
  • posting_preview.tpl
Find select 
<span class="postbody">{MESSAGE}</span>
Replace With select 
<div class="postbody">{MESSAGE}</div>

Posting Topic Review Edits

Some versions of Internet Explorer do not handle nested iframes well, which is the primary method of audio and video embeds... So a simple edit to remove the iframe and replace with a fixed height div.


  • yoursite.com
  • modules
  • Forums
  • templates
  • subSilver
  • posting_topic_review.tpl
Find select 
<iframe width="100%" height="300" src="{U_REVIEW_TOPIC}">
Replace With select 
<div id="ftopic-review">
Find select 
</iframe>
Replace With select 
</div>
