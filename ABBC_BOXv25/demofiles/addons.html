Optional Addons

Take a moment to look over the selection of optional addons which should be installed prior to proceeding to the final step of installation. This assumes you have completed step one and step two of installation and the mod is working in all active themes. There are a handful of troubleshooting tips available in the FAQ if you are having issues.

Code Highlighting featuring Highlight

16 languages are supported by default, although over 40 languages can be used if required by an individual site. The basic 16 valid classes are: apache, bash, cpp, cs, css, diff, html, ini, java, javascript, perl, php, python, ruby, sql, xml

Install

Upload all files within ABBC_BOXv25/extras/Highlighter/root to yoursite.com/
  • yoursite.com
  • includes
  • addons
  • head-abbcbox.php
Find select
$UseCodeHighligher = false; // enables code highlighting
Change To select
$UseCodeHighligher = true; // enables code highlighting

At this point code highlighting should be functional with the default highlight style. To use a custom highlight style for your theme, simply rename the desired style (located at includes/jquery/styles/) to highlight.css and upload to the style directory of your theme. Note that if your highlighting style uses images (a couple do) make sure to adjust image paths within the newly created highlight.css accordingly (required images are also located at includes/jquery/styles/).

If you are using either the Blue_Blog and/or CT_RN themes, an example highlight.css file for those themes is provided at:
ABBC_BOXv25/extras/Highlighter/Theme_Specific_Styles/themes

Note that the other default RN themes all use the default highlighting style. See Customization for more tips regarding code highlighting; as well as some important notes regarding browser compatibility and styling.

Iconize Links to Desired Sites

A purely CSS based addon which automatically adds icons to links, allowing for specific icons to be used for specific sites. Use the included Vector Social Media Icons to "Iconize" as many popular social sites as desired; or create/find your own icons to "Iconize" your favorites sites from around the web. Easy to add sites and customize to your needs. Intended for use in the forums, but can be used sitewide if desired. Note that some tweaking of line-height or font-size may be required, and that 16px icons will not work well if your font-size is too small.

Install

CSS declarations (and minor edit) need to be applied to all copies of bbcode_addons.css which you have uploaded to your site. The number of copies will vary depending on how many themes you have installed and their configuration. Note there is a slight difference to the edit if applying to themes/ANY_THEME/style/bbcode_addons.css (option a) as opposed to mods/bbcode_box/bbcode_addons.css (option b); as the path to the image files needs to be adjusted. If you want to apply this sitewide be aware these stylesheets only load within the Private Message and Forums modules, so you would need to move code to your style.css. You also must use class postlink on all links you want icons on, read the note below for more info.

Easy YouTube Video Embeds

Let's face it, users are lazy or don't necessarily always understand how to use bbcode. So why not allow for easy embedding of youtube videos by simply including the URL to the video, with no extra markup required? Enter the EasyTube Mod to add auto-embed support for over a dozen popular youtube URL's. Special thanks to eldorado for the idea and getting the ball rolling on implementation, and to ridgerunner for writing and maintaining an excellent regular expression suited for the job.

Install

  • yoursite.com
  • modules
  • Forums
  • includes
  • bbcode.php

PRE-EDITED file available at extras/EasyTube/bbcode.php which includes ABBC Mod plus EasyTube

Find select
function make_clickable($text)
{
	$text = preg_replace('#(script|about|applet|activex|chrome):#is', "\\1&#058;", $text);

	// pad it with a space so we can match things at the start of the 1st line.
	$ret = ' ' . $text;
After Add (on a new line) select
	
	// look for youtube urls and create video embeds from them (by ridgerunner)
	// http://stackoverflow.com/questions/5830387/php-regex-find-all-youtube-video-ids-in-string
	// ** modified for use in forums
	$ret = preg_replace('~
	# Match non-linked youtube URL in the wild. (Rev:20111012)
	(^|[\n ])         # match space or newline at start**
	(?:https?://)?    # URL scheme. Either http or https.**
	(?:[0-9A-Z-]+\.)? # Optional subdomain.
	(?:               # Group host alternatives.
	  youtu\.be/      # Either youtu.be,
	| youtube\.com    # or youtube.com followed by
	  \S*             # Allow anything up to VIDEO_ID,
	  [^\w\-\s]       # but char before ID is non-ID char.
	)                 # End host alternatives.
	([\w\-]{11})      # $1: VIDEO_ID is exactly 11 chars.
	(?=[^\w\-]|$)     # Assert next char is non-ID or EOS.
	(?!               # Assert URL is not pre-linked.
	  [?=&+%;\w]*     # Allow URL (query) remainder.**
	  (?:             # Group pre-linked alternatives.
		[\'"][^<>]*>  # Either inside a start tag,
	  | </a>          # or inside <a> element text contents.
	  )               # End recognized pre-linked alts.
	)                 # End negative lookahead assertion.
	[?=&+%;\w]*       # Consume any URL (query) remainder.**
	~ix', 
	'\\1<div class="forumvideo video"><iframe src="http://www.youtube.com/embed/\\2" frameborder="0" allowfullscreen></iframe>
<div class="forumvideotagline"><a class="postlink youtu ficon" href="http://youtu.be/\\2" target="_blank">youtu.be/\\2</a></div></div>',
	$ret);

At this point you can upload and the mod should be working, although there is one customization you may want to make first; choosing the size and position of these "EasyTube" videos. To do so, change the class of the <div> that wraps the iframe to one of the following classes: video, video-small, video-medium, video-large, video-left, video-right, video-center, video-small-left, video-small-right, video-small-center, video-medium-left, video-medium-right, video-medium-center, video-large-left, video-large-right, video-large-center.

Tricked Out News #hashtag Mod

Adds a Twitter like hashtag mod for your RavenNuke™ forum. If you are not familiar with hashtags, they are any SINGLE word that you place the # sign in front of, to link to a keyword search in the forum. Although it functions in a similar manner to that of the newly added [tag]term[/tag] it makes a nice shorthand compliment for quickly tagging desired posts. Not to mention, installation only requires one additional line of code!

Install

  • yoursite.com
  • modules
  • Forums
  • viewtopic.php

PRE-EDITED file available at extras/Hashtag/viewtopic.php

Find select
 $message = $postrow[$i]['post_text'];
After Add (on a new line) select
 $message = preg_replace('/(^|\s)#([a-z0-9_-]{3,}+)/i', '$1<a href="modules.php?name=Forums&amp;file=search&amp;search_keywords=$2&amp;mode=results">#$2</a>', $message);
